This project uses a public database, saved locally within the former_names.csv, goalscorers.csv, results.csv, and shootouts.csv. 
The main.py file contains many lines which have been commented out as they were for creating data, since saved locally within advancement_probabilities.csv, advancement_probabilities.png, cleaned_data.csv, elo_ratings.csv, match_results.png, and monte_carlo_fixtures.csv
If cloning files, uncomment sections based on what data needs to be updated.
This model was trained using a combination of a Random Forest model and Poisson Model for score generation
